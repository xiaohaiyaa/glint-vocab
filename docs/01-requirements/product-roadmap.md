# Glint(拾光) --录入单词复习 产品需求文档(PRD)

> 文档版本: v0.1 | 编写目的: 明确开发边界范围，防止需求蔓延

## 1.项目定位
- 一个 **录入即复习** 的语境单词本。用户通过手动输入或者粘贴遇见的生词和原句，智能匹配相关语境句子（若粘贴原句则没有这一步），系统通过时间间隔帮助用户在语境句子中掌握词汇。

## 2.目标用户
- 正在阅读英语原著、外刊，希望有一个 **干净的自己导入式学习生词** 的非母语者英语学习者。

## 3.解决痛点
- 现有的大多数App以 **导入词库** 方式学习单词，不匹配小部分人学习方式，此产品通过此方式弥补需要通过这种方式学习的方式;
- 用户自己记录的笔记太过散乱，甚至出现 **记下即放下** 的记忆方式。此产品以统一方式输出，并融合复习机制给出每天需要解决的记忆单词。

## 4.v0.1功能清单
- [  ] **录入功能**: 必须同时输入“单词”和“例句”才能保存。如果用户只输入单词，保存按钮置灰（或提示“请粘贴你遇到它的原句”）;
- [  ] **复习卡片**: 翻页展示待复习的句子（生词默认隐藏），点击显示释义;
- [  ] **反馈机制**: 复习卡下方有三个分类按钮：“陌生”、“模糊”、“掌握”;
- [  ] **首页统计**: 显示“今日录入单词”“需要复习X个单词”;

## 5.后续版本待做功能清单
- [  ] **登录注册**: 云端存储，此版本数据存本地;
- [  ] **智能匹配**: 基于海量词库输入单词可以智能匹配所需的语境句子;
- [  ] **语音朗读**;
- [  ] **拼写测试**;
- [  ] **手机端App**;

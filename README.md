# Glint（拾光）

> **一个“录入即复习”的语境单词本**

用户录入生词和原句，系统通过间隔算法帮助用户在语境句子中掌握词汇。

---

## 🎯 项目定位

**不是背单词，是泡单词。**

你录入的每一个生词，都会在被你遗忘之前，通过你亲手录入的例句再次出现。系统不打扰你阅读，只在合适的时机，用你熟悉的语境唤醒记忆。

---

## ✨ 核心特性

- **📝 语境录入**：必须同时输入“单词”和“例句”才能保存，强制养成“带语境背单词”的习惯
- **🔁 间隔复习**：基于三档反馈（陌生/模糊/掌握）自动计算下次复习时间
- **🎯 语境驱动**：复习时只展示例句，生词默认隐藏，逼用户靠语境猜词义
- **🧠 自动翻译**：录入时自动调用翻译 API 填充释义，用户可手动修改
- **📊 首页统计**：今日录入数 / 待复习数，一目了然

## 🚀 快速开始

```bash
# 克隆项目
git clone git@github.com:xiaohaiyaa/glint-vocab.git
cd glint-vocab

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 `http://localhost:5173` 即可开始使用。
